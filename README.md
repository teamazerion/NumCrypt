**Secret Numeric Language Tool for Terminal**

NumCrypt is a lightweight terminal-based tool that converts normal text into a secret numeric code and decodes it back using a private key. It acts like a simple custom language that only users with NumCrypt can understand.

Built for privacy, experimentation, and fun with cryptography concepts.

---

## Features
- Encode text into secret numbers  
- Decode numbers back into original text  
- Uses a private numeric key  
- Works fully offline  
- Simple CLI interface  
- Runs on Termux / Linux / Python  

---

## Use Cases
NumCrypt is useful for:
- Students learning cryptography basics  
- Developers experimenting with encoding systems  
- Secret messaging (basic level)  
- Cybersecurity enthusiasts  
- Fun custom language projects
